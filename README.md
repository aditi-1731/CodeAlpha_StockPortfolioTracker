# Stock Portfolio Tracker

## Description

A simple Python application that allows users to calculate the total value of their stock investments using predefined stock prices. The program validates user input, calculates the investment amount, displays a summary, and saves the results to a text file.

## Features

- Display available stocks and their prices
- Validate stock names entered by the user
- Calculate investment value for multiple stocks
- Display total portfolio value
- Save portfolio summary to a text file
- Modular code using Python functions
